###docker에서 gitlab-ce, gitlab-runner,linux 이미지 다운로드
##gitlab 서버 설치
docker run --detach --hostname gitlab.example.com --publish 4443:443 --publish 8888:80 --publish 2222:22 --name gitlab --restart always --volume /srv/gitlab/config:/etc/gitlab --volume /srv/gitlab/logs:/var/log/gitlab --volume /srv/gitlab/data:/var/opt/gitlab gitlab/gitlab-ce:latest

 ##gitlab-runner시스템설치
docker run -d --name gitlab-runner --restart always -v /srv/gitlab-runner/config:/etc/gitlab-runner -v /var/run/docker.sock:/var/run/docker.sock gitlab/gitlab-runner:latest

1. gitlab에서 프로젝트 생성(빈프로젝트) 
2. spring boot project를 gitlab의 프로젝트와 연결하기 
3. spring boot project폴더 gitlab-ci.yml을 작성(docker의 웹서버에 jar파일을 업로드하고, jar을 동작시켜 웹서버가 동작되는 배포까지 처리하는코드 작성 
4. push를 할 경우 최종 배포 완료 를 전부 상세하게 설명해줘
